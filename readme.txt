Display lunch menu from 'orderlunchesatwork.com'.

== Description ==

This widgets rips the HTML from the targeted Order Lunches At Work menu and uses that to find and display what is for lunch today.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Get the variables from the URL by going to *your_subdomain*.orderlunchesatwork.com "Order" > "View Menu"
4. Plug URLs in to widget.
